Broadband Map Quality Assessment Matches

About

This application displays June 2012 National Broadband Map data matches to Baker Quality assessment data


Details:

  1. Make selections on left side of screen
  2. Matches by both "Provider" and "Download Speed" for each provider.
  3. WIRELINE PROVIDERS: Comcast, AT&T, Frontier, Verizon
  4. WIRELESS PROVIDERS: Verizon, AT&T, Sprint, T-Mobile

Data Source:
National Broadband Map (NBM); Data as of June 2012
